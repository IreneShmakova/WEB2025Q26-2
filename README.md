# WEB2025Q26-2
Репозиторий для Лабораторной работы №2, курс "Web-программирование" 2025-2026.

В рамках данной работы была поставлена задача создать todo-лист. 

Ползователь должен иметь вохможность:
- создавать, редактировать и удалять задачи,
- отмечать задачи как выполненные,
- фильтровать, сортировать и искать задачи по дате и названию,
- менять порядок задач с помощью drag-and-drop.
Также должно быть реализовано сохранение текущего состояния в памяти браузера.
